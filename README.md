# Harmish Patel | Personal Portfolio

Personal portfolio made using React, Material-ui and Github pages.
[Demo](https://harmishpatel21.github.io/)


## Commands to run and deploy on github pages.

In the project directory, you can run:

### `git clone [repo]`

Clone it and make changes accordingly.

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

### `git commit -am "[type your commit message]"`

Commit the changes to github repo

### `git add .`

Add changes to repo.

### `git push`

Push the changes.

### `npm run deploy`

It will deploy your changes and site to github pages repo.

Try it with [YourRepo.github.io]
